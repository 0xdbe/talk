Software architecture

The components of a modern websites are:

  • Front-End (Client side)

  • Back-End (Server side)


Full Stack developer: someone who is able to work on both the front-end and back-end component
  • Front-End est un composant qui s'exécutera dans le navigateur de l'utilisateur
  • Back-end est ensemble de composants qui s'exécutent sur les serveurs (Généralement REST API)
  • Full stack: difficile d'avoir les compétences dans les 2 domaines.

Front-End component


  • SPA (Single Page Application)
  • Set of files (HTML, CSS and JS)
  • Responsive design (Desktop and mobile)


Building an SPA is becoming increasingly difficult !

  • SPA : Qui connais ? Et le contraire => MPA (Multi Page Application)
  • Retour des RIA (Rich internet application) que nous avons connu avec Flash ou encore les applets Java

How to develop your SPA?

  • with a native browser language compliant with ECMAScript
    • Javascript
    • ActiveX
    • Action Script
    • ...
  • With a basic framework (JQuery, RxJS, Promise, ...)
  • With an advanced framework (Angular, ...)
  • "simple" et "advanced" sont des termes personnels pour distinguer la différence entre les 2 types de framework
  • Evolution : 1999 = JavaScript / 2005 = simple framework / 2010 : advanced framework
  • Chaque niveau s'appuie sur le précédent

What's an advanced framework such as Angular?

  • A software design pattern (MVC)
  • A language (Type Script)

  • Development tools:
    • NodeJS and NPM are needed
    • Angular tools (Angular CLI, Angular Compiler, ...)
    • webpack (module builder)
    • Karma (unit testing)
    • ...
    • NodeJS = équivaut à JVM, Pyhton, …
    • NPM = équivaut à PIP en python ou PPM pour PERL
    • module builder = génération des fichiers JS (module)

How to deploy my SPA with Angular?

  • Transpiling:
    • Just-in-Time (JiT)
    • Ahead-of-Time (AoT)

    And many other tasks should be done...

    • Jit: source téléchargé puis compiler via le navigateur (pour la dev)
    • AOT : mise à disposition du code compilé (plus rapide et plus léger, pour la production)
